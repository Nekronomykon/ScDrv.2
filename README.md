# ScDrv

A cross-platform tool for chemists with computers. 

## Fields: 
* molecular modeling
* conformational analysis
* structural chemistry
* quantum chemistry
* computational chemistry
* molecular graphics
* numerical analysis

## Depends on:
* [VTK](http://vtk.org) (current version 8.2.0)
* [Qt](http://qt.io) (current version 5.12.2)
